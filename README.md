Inspiration
- I wanted to create an Robotarm as an Project for my class and will use 5 Servo motors and a STM32 Nucleo64 F446RE Microcontroller and a Joystick

Challenges
- Routing the Jumpercables through the actual arm and making the arm spin and not riping out the cables
